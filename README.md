# Matthew Pizzitola Personal Portfolio Website

This repository contains the source code for my personal portfolio website, built to showcase my background, technical skills, project work, and ongoing learning as a recent B.A. Honours Information Technology graduate focused on entry-level IT Support, Help Desk, systems, networking, web, database, and technical documentation roles.

## Live Website

[View Portfolio](https://matthew5500.github.io)

## Overview

The website is a responsive single-page portfolio built with HTML, CSS, and JavaScript. It includes sections for my background, focus areas, technical projects, project journal, contact information, social links, and downloadable resume.

The goal of this portfolio is to present my work in a clear, professional, and accessible format while demonstrating practical front-end development, UI organization, project documentation, and form integration skills.

## Features

* Responsive single-page layout
* Fixed navigation with mobile menu support
* About section with skills, experience, and education tabs
* Focus cards for IT Support, Systems & Networks, and Web/Data Projects
* Interactive project gallery with flip-card details
* Expandable project image previews
* Searchable Projects section
* Journal section with timeline-style project updates
* Contact form connected to Google Sheets through Google Apps Script
* Email notification support for new contact form submissions
* Honeypot spam protection and front-end validation
* Reduced-motion accessibility toggle

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Apps Script
* Google Sheets
* GitHub Pages
